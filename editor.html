🖱️ 기본 조작

드래그화면 이동 휠화면 스크롤 Ctrl+휠확대/축소 클릭요소 선택 R화면 초기화 Del선택 삭제 ESC취소/해제

✏️ 편집 모드 E

포트와 블록을 이동하고 할당

드래그포트/블록 이동 클릭→클릭포트를 블록에 할당 더블클릭새 포트 생성 Shift+더블새 블록 생성

🔗 연결 모드 C

요소들 사이에 연결선 그리기

클릭→클릭시작 → 끝 연결 빈공간꺾임점 추가 Shift+클릭대각선 (직교 무시) Alt+클릭기존 선에서 분기

📋 복제 모드 K

부모 영역의 연결을 자식에 복제

① 부모 영역 드래그 (녹색)
② 자식 영역 드래그 (파란색)
③ Shift+클릭 적용

📐 템플릿 모드 T

블록-포트 배치 패턴 저장/적용

① 기준 블록 클릭
② 드래그로 포트들 선택
③ 사이드바에서 적용

🔍 패턴 모드 P

연결 패턴 저장 및 유사 검색

① 드래그로 영역 선택
② 패턴 저장
③ 유사 그룹 검색/적용
